All textbooks created with this shell are published under a Creative Commons license.

**Why:** User explicitly stated this preference; do not add traditional copyright notices (© All rights reserved).
**How to apply:** Use a Creative Commons attribution notice instead of © copyright on splash pages, footers, and any other pages that reference rights. Ask user which CC license variant (e.g., CC BY, CC BY-NC, CC BY-SA) if not specified.
